# Lavalink Setup Files

These files are ready to deploy to WispByte.

## Quick Start

1. Upload this entire folder to WispByte
2. Run: `docker-compose up -d`
3. Get IP: `curl ifconfig.me`
4. Update `music.py` with the IP
5. Restart your bot

## Files

- **docker-compose.yml** - Docker configuration
- **application.yml** - Lavalink settings (change password!)
- **README.md** - This file
